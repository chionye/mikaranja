
# Simple Todo List

A simple CRUD todo list app with dummyJSON.


## Run Locally

Clone the project

```bash
  git clone https://github.com/chionye/mikaranja.git ./
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```


## Demo

Navigate to http://localhost:3000 and login with credentials.
